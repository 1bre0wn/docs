La sección de **Plantillas de Correo** te permite gestionar todas las comunicaciones que tus usuarios reciben.

## ¿Qué puedes hacer con las plantillas?

* **Personalización total:** Adapta los correos con el logo y colores de tu organización.
* **Variables dinámicas:** Inserta automáticamente el nombre del empleado.
* **Previsualización real:** Comprueba cómo se ve el correo.

## Uso de Variables Dinámicas

| Variable | Resultado al enviar |
| :--- | :--- |
| `{{.FirstName}}` | El nombre del empleado. |
| `{{.LastName}}` | El apellido del empleado. |
| `{{.Email}}` | El email del empleado. |
| `{{.URL}}` | La url del phishing. |

<Tip>
  Usa `{{.FirstName}}` en el **Asunto** del correo para aumentar la tasa de apertura.
</Tip>
